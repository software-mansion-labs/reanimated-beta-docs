# useFrameCallback

`useFrameCallback` lets you run a function on every frame.

## Reference

```javascript
import { useFrameCallback } from "react-native-reanimated";

function App() {
  // ...
}
```

- The callback passed to the 1st argument is automatically [workletized](/docs/fundamentals/glossary#to-workletize) and ran on the [UI thread](/docs/fundamentals/glossary#ui-thread).
